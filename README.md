# AgentQuote

AgentQuote is a static product prototype for choosing the best AI agent route before a task starts.

## GitHub Pages

This folder can be published directly with GitHub Pages:

1. Create a new GitHub repository named `agentquote`.
2. Upload the files in this folder to the repository root.
3. In GitHub, open Settings -> Pages.
4. Set Source to `Deploy from a branch`, Branch to `main`, Folder to `/root`.
5. The page will be available at `https://leah1314.github.io/agentquote/`.
